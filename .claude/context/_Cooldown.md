# Cooldown

App gratuito de barra de menus (macOS) que avisa quando o limite de uso de uma IA (Claude, ChatGPT, Gemini...) reseta. Desenvolvido pela Salto Solutions como vitrine/canal de contato.

**Bundle id:** `solutions.salto.cooldown` · **macOS mínimo:** 14 (Sonoma) · **Idiomas:** pt-BR / en-US (padrão inglês)

## Por onde começar
- [[produto]] — presets de serviço, decisões de produto
- [[lancamento]] — plano/checklist de lançamento

## Stack
Swift + SwiftUI (`MenuBarExtra`, app agente) · Swift Package Manager (sem Xcode) · `UserDefaults` + `Codable` · Liquid Glass nativo (macOS 26+)
